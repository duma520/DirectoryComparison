## 1. 工具简介

**多目录文件比对工具** 是一款开源、跨平台的图形化工具，用于快速比较两个目录下的文件差异。  
它不仅能告诉你 **“哪些文件不一样”**，还能显示 **文件大小、修改时间**，并用 **三种颜色** 直观标识文件状态。

### 🎯 适合谁用？
- **普通用户**：备份文件夹前后对比、整理照片、检查重复文件。
- **开发者**：对比项目代码版本、检查部署文件一致性。
- **运维人员**：验证服务器配置文件、同步日志目录。
- **设计师/文案**：对比设计稿版本、文档修改记录。
- **教育/学生**：理解文件系统、学习编程中的目录遍历。

### ✨ 核心功能
- 快速扫描两个目录（支持子目录递归）
- 彩色标识：共同存在 / 多出 / 缺少
- 5种过滤模式 + 3种显示模式
- 历史记录 + 拖拽支持
- 多线程扫描，界面不卡顿
- 右键菜单快速操作文件

## 3. 快速上手（3分钟入门）

### 第一步：选择目录
- 点击 **“浏览”** 按钮选择两个文件夹。
- 或直接 **拖拽文件夹** 到输入框。
- 也可以使用 **“粘贴”** 按钮从剪贴板粘贴路径。

### 第二步：开始比较
- 点击工具栏的 **“开始比较”**（或按 F5）。
- 等待扫描完成（大目录会显示进度条）。

### 第三步：查看结果
- 结果会按 **颜色** 显示：
  - 🟢 薄荷绿：两个目录都有的文件
  - 🟠 蜜桃橙：目录2中缺少的文件
  - 🌸 樱花粉：目录1中多出的文件

### 第四步：过滤与切换视图
- 使用 **“比较模式”** 下拉框筛选显示内容。
- 使用 **“显示模式”** 切换分组/平铺/列表视图。

### 第五步：操作文件
- 右键点击文件 → 打开位置 / 复制路径 / 复制文件名。

## 5. 五种比较模式详解

| 模式名称 | 显示内容 | 适用场景 |
|----------|----------|----------|
| **全部文件** | 显示所有文件和目录 | 完整对比，适合初次检查 |
| **只显示共同存在** | 两个目录都有的文件 | 寻找同名文件，验证同步状态 |
| **只显示差异** | 只显示存在差异的项目 | 快速定位不一致内容 |
| **只显示多出的文件** | 目录1有，目录2没有 | 检查新增或备份遗漏 |
| **只显示缺少的文件** | 目录2有，目录1没有 | 检查被删除或丢失的文件 |

## 6. 三种显示模式详解

### 6.1 分组显示（默认）
- 文件按状态分组：**共同文件 / 仅在目录1 / 仅在目录2**
- 每个分组折叠显示，组内显示完整信息（大小、时间）
- ✅ **适合：整体概览、分类处理**

### 6.2 平铺显示
- 所有文件平铺展开，显示全部列信息
- 无分组，便于排序和批量操作
- ✅ **适合：排序查看、导出对比**

### 6.3 列表显示
- 只显示文件名和状态背景色
- 隐藏大小、时间等次要信息
- 类似资源管理器，清爽简洁
- ✅ **适合：快速浏览、大目录概览**

## 8. 各行业应用场景举例

### 👨‍💻 软件开发
- **场景**：对比本地代码与服务器代码
- **操作**：选择本地项目目录和服务器解压目录 → 点击比较 → 查看“多出的文件”找出新增功能

### 🗄️ 数据备份
- **场景**：验证备份是否完整
- **操作**：选择原数据目录和备份目录 → 只显示“缺少的文件” → 确认无遗漏

### 📸 摄影师/设计师
- **场景**：整理多版本照片/设计稿
- **操作**：选择两个版本文件夹 → 分组显示 → 快速定位修改过的文件

### 🧑‍🏫 教育培训
- **场景**：教学文件系统遍历
- **操作**：让学生用工具对比两个不同结构的文件夹，理解递归遍历原理

### 📄 文档管理
- **场景**：对比两份合同/报告
- **操作**：右键复制路径 → 粘贴到比较工具 → 直接查看同名文件差异

## 9. 高级技巧与自定义

### 9.1 快速交换目录
点击 **“交换目录”** 按钮，可以快速互换两个目录路径，适合反复对比。

### 9.2 使用历史记录
每次选择的目录会自动保存（最多20条），点击工具栏 **“历史记录”** 可快速复用。

### 9.3 拖拽支持
直接从文件管理器拖拽文件夹到输入框，无需手动浏览。

### 9.4 配置文件保存
工具会自动保存：
- 上次使用的两个目录
- 当前选择的比较模式与显示模式
- 窗口位置与大小
- 历史记录列表

### 9.5 颜色自定义（进阶）
你可以修改源码开头的颜色定义部分，替换为自己的配色：
```python
COLOR_EXIST = QColor('#B5EAD7')   # 薄荷绿
COLOR_MISSING = QColor('#FFDAC1') # 蜜桃橙
COLOR_EXTRA = QColor('#FFB7CE')   # 樱花粉
```

## 10. 常见问题（FAQ）

### ❓ 为什么我的大目录扫描很慢？
- 工具采用多线程扫描，但首次遍历大目录仍需要时间。请耐心等待，或使用 **“停止”** 按钮取消。

### ❓ 为什么有些文件没有显示大小/时间？
- 目录本身没有大小；如果文件无权限读取，也会显示为 `-`。

### ❓ 能否比较文件内容是否相同？
- 当前版本仅比较文件名和存在性。内容比较功能为预留项，欢迎开发者贡献。

### ❓ 是否支持网络路径？
- 支持，只要系统能访问的路径（如 `\\server\share`）都可以输入。

### ❓ 如何报告问题或建议？
- 请访问 [GitHub Issues](https://github.com/duma520) 提交，我们会在公开渠道回复。

## 11. 开发者扩展指南

### 项目结构概览
```
DirectoryComparison/
├── main.py               # 主程序入口
├── ProjectInfo.py        # 项目元数据
├── dialogs/              # 对话框类
├── threads/              # 扫描/比较线程
├── widgets/              # 自定义控件
└── resources/            # 图标等资源
```

### 如何添加新的过滤模式？
1. 在 `CompareMode` 枚举中添加新成员。
2. 在 `filter_results()` 中添加对应逻辑。
3. 在菜单和下拉框中添加选项。

### 如何修改显示列？
修改 `self.result_tree.setHeaderLabels()` 中的列表即可。

### 打包为独立可执行文件
推荐使用 `PyInstaller`：
```bash
pip install pyinstaller
pyinstaller --onefile --windowed --icon=icon.ico main.py
```
